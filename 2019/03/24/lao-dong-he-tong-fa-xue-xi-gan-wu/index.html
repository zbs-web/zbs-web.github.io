劳动合同法学习感悟

  1. 劳动合同法适用人群：
    • 企业
    • 个体户
    • 非企业单位
    • 与劳动者建立劳动关系的单位
  2. 劳动规章制度合法有效的条件：经过民主程序公示，并告知劳动者
  3. 工会的作用：帮助劳动者签订劳动合同，指导用人单位履行劳动合同，维护劳动者合法权益
  4. 用人单位自用工之日起就与劳动者建立了劳动关系
  5. 用人单位在建立劳动关系过程中应该建立职工名册
  6. 用人单位招收劳动者，不能收取劳动者押金
  7. 用人单位应在用工前 / 用工之日 / 用工之日起一个月内与劳动者签订劳动合同
  8. 劳动合同的类型
    • 固定期限
    • 无固定期限
    • 以完成一定任务为期限
  9. 订立无固定期限劳动合同的情况：
    • 劳动者连续工作满10年
    • 在企业初次实行劳动合同制 / 国有企业改制 的条件下，劳动者连续工作满10年且距退休不足10年
    • 连续两次订立劳动合同后的续订
  10. 用人单位满1年不与劳动者订立书面劳动合同 / 超过1个月且不满1年，视为已经订立无固定期限劳动合同。后者用人单位在这段时间内须向劳动者每月支付二倍工资
  11. 劳动合同不满3个月，不得约定试用期
  12. 试用期最长不超过6个月
  13. 试用期时间划分：
  • 3个月 <= time < 1年 ：1个月
  • 1年 <= time < 3年 ：2个月
  • >= 3年 ：6个月
  1. 试用期工资不得低于劳动合同约定工资的80%
  2. 用人单位可以将下列人员列为竞业受限人员：
  • 高级管理人员
  • 高级技术人员
  • 负有保密义务的人员
  1. 用人单位可以与劳动者约定由劳动者承担违约金的情况
  • 用人单位提供培训，并约定服务期
  • 用人单位与负有保密义务的劳动者签订有竞业限制条款
  1. 劳动合同无效的情况：
  • 采用欺诈、胁迫的手段
  • 用人单位免除自己的法定责任，去除劳动者权利
  • 违反法律规定
  1. 当用人单位发生分立、合并时，原劳动合同继续有效，由继承原单位权利与义务的新用人单位继续履行
  2. 劳动者提前30日（试用期为提前3天）向用人单位提出书面解除劳动合同，用人单位无需支付经济补偿金
  3. 劳动者患病在医疗期满后不能继续在该单位工作，用人单位应提前30日书面通知 / 支付1个月工资后解除劳动合同
  4. 劳动者患病，在医疗期内劳动合同到期，劳动合同则应延续到医疗期满后再终止
  5. 经济补偿支付标准（以工作年限为标准）
  • >= 1年 ：支付1个月工资
  • 6个月 <= time < 1年 ：同上
  • < 6个月 ：支付半个月工资
  1. 劳动者可以不通知用人单位立即解除劳动合同的情况
  • 用人单位违规指挥，危害劳动者人身健康
  • 用人单位以暴力、非法手段胁迫劳动者进行劳动
