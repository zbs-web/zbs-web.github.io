互联网创业大赛参赛指南

赛前热身

  1. 重点：商业性 / 经济效应
  2. 比赛的意义：获得职业能力的提升

参赛准备

  • 优秀的参赛作品的基本要素
    • 好的项目
      • 【痛点】符合市场需求（解决痛点）
      • 【尖叫点】创新性 + 科学性（高大上、深奥）
      • 【引爆点】应用前景，市场能切入
        • 应思考项目如何能切入市场
        • 例如：手机业（南京大学）、广告业（东南大学）
      • 【想象空间】市场前景
      • 【战略规划】可持续发展 + 赢利
    • 好的团队
  • 准备阶段需要做什么？
    • 找准市场需求，客户痛点，从而选择一个好项目
    • 创业领域选择（三度交集法）
      • 高财富
      • 高擅长
      • 高兴趣
    • 组建好团队的标准
      • 志同道合（有一个共同的目标）
      • 情投意合（能做到一起去）
      • 能力匹配（不同领域的人做不同的事）
      • 【注】团队组建在总分值中有40分（如何考核呢？考核的标准是什么？）
        • 基于项目需求的人力规划 / 岗位设计
        • 基于创业需求的能力匹配（不同的人对应不同的岗位）
        • 基于企业发展规划的激励设计
    • 研发好的产品（好产品的标准是什么？）
      • 国际 / 国家专利
      • 软件著作权
      • 学术论文
      • 科研成果鉴定（四川省科技厅负责）
      • 专业赛事大奖
      • 使用证明 / 委托开发文件 / 销售合同
      • 【注】最好有1-2个
